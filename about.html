    Fichiers journaux

    JstMStudio suit une procédure standard d'utilisation des fichiers journaux. Ces fichiers enregistrent les visiteurs lorsqu'ils visitent des sites Web, des logiciels, des applications et des services. Toutes les sociétés d'hébergement, qu'elles soient payantes ou gratuites, le font et font partie de l'analyse des services d'hébergement. Les informations collectées par les fichiers journaux incluent les adresses IP (protocole Internet), le type de navigateur, le fournisseur d'accès Internet (FAI), l'horodatage, les pages de renvoi/sortie, le type d'appareil, l'emplacement et éventuellement le nombre de clics. Ceux-ci ne sont liés à aucune information personnellement identifiable. Le but de l'information est d'analyser les tendances, d'administrer le site, de suivre les mouvements des utilisateurs sur le site Web, de suivre l'utilisation des sites Web, des logiciels, des applications et des services et de collecter des informations démographiques.

    Droits de protection des données GDPR

    Si vous êtes dans la zone effective du RGPD, JstMStudio souhaite s'assurer que vous êtes pleinement conscient de tous vos droits de protection des données accordés dans les zones de l'UE. Chaque utilisateur dispose des droits de protection des données suivants :

    • Le droit d'accès - Vous avez le droit de demander des copies de vos données personnelles. Nous pouvons vous facturer une somme modique pour ces services.
    • Le droit de rectification : vous avez le droit de nous demander de corriger toute information que nous avons collectée et que vous jugez inexacte. Vous avez également le droit de nous demander de compléter les informations que vous jugez incomplètes.
    • Le droit à l'effacement : vous avez le droit de nous demander d'effacer vos données personnelles, sous certaines conditions.
    • Le droit de restreindre le traitement : vous avez le droit de demander que nous restreignions le traitement de toutes vos données personnelles, sous certaines conditions discutées.
    • Le droit de vous opposer au traitement - Vous avez le droit de vous opposer à notre traitement de toutes vos données personnelles, sous certaines conditions discutées avec vous.
    • Le droit à la portabilité des données : vous avez le droit de demander que nous transférions les données que nous avons collectées à une autre organisation, ou directement à vous, sous certaines conditions.

    Si vous faites une demande à l'équipe, nous avons un délai d'un mois pour vous répondre. Si vous souhaitez exercer l'un de ces droits, veuillez nous contacter directement.
